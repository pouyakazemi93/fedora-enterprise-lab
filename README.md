#Fedora Enterprise Lab
This project demonstrates real-world Linux server configuration in Virtualized enviroment.

##Project Overview

This Lab includes:

1.  Multi-disk configuration
2.  LVM setup and management 
3.  BTRFS filesystem usage 
4.  Multiple IP configuration
5.  Apache Web Server with 2 Virtual Hosts
6.  FTP Server setup
7.  SSH configuration 
8.  Local RPM repository creation
9.  Package management using dnf and rpm
10. User and Group management 
11. Permission management
12. System monioring (btop)
13. File management (mc)

-------------------------------------------------------------------------------------------
##Architecture 

OS:Fedora Linux (Virtual Machine)
Virtualization:VMWare
Web Server:httpd
FTP:vsftpd
Repository:Lpic 
Storage:LVM+BTRFS
